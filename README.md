# **boole-js**

Library to solve Boolean exercises

## Instructions to develop

1. Install the dev dependencies:

    ~~~
    npm i
    ~~~

2. Install globally:

    - Webpack
        ~~~
        npm i -g webpack
        ~~~

    - Webpack-cli
        ~~~
        npm i -g webpack-cli
        ~~~

    - Jest
        ~~~
        npm i -g jest
        ~~~

## To execute tests

Run the next command line:
~~~
jest
~~~
